# 福彩3D百十个杀码预测 v8

每期预测百位、十位、个位各杀一码，全自动云端更新部署。

## 算法
- **v8架构**: v5条件分支 + 集成默认投票(3公式取众数) + 增强回退(10备选)
- **近100期准确率**: 99.000%（百位98% 十位99% 个位100%）
- **安全保证**: 每期仅用上期数据，不偷看未来，无参数调优

## 自动更新
- GitHub Actions 每日3次自动抓取（北京 22:00 / 23:30 / 01:00）
- 多数据源fallback保障
- 自动部署到 GitHub Pages
